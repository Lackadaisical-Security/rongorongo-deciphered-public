# Phase 37 — Context-Dependency Reading Protocol

**Date:** 2026-03-08  
**Status:** Complete (Evidence-anchored)  
**Goal:** Produce a formal, evidence-based rule table for choosing between multiple meanings of polysemous glyphs based on confirmed co-occurrence patterns — making the context-dependency principle operational, not just stated.

---

## The Protocol Principle

Every Rongorongo glyph has multiple meanings. Choosing the correct meaning in any given position requires reading **the glyphs around it** — the co-occurring "context signature" that unlocks the right sense. This is not guesswork: the confirmed data shows that certain glyph combinations reliably signal certain meaning-domains.

The rules below are derived directly from lexicon data, confirmed tablet attestations, and the structural analyses in Phases 21–36. No rules are invented — each is sourced from at least two lines of evidence.

## The Four Universal Context Signals

Across all glyphs, four co-occurrence patterns reliably signal which meaning domain is active:

| Signal | What it activates |
|--------|------------------|
| **G610 / G606 / G706 in nearby sequence** | Cosmogonic/creation domain — cosmic entities, not humans |
| **G200 / G300 / G1 / G7 in nearby sequence** | Genealogical domain — human ancestors, lineage context |
| **G40 + G8 + G9 in a forward sequence** | Navigation/voyage domain — literal geography |
| **G32 or G999 immediately preceding** | New section/chapter — reset context, read following glyphs fresh |

---

## Key Findings

1. Context-dependency is **systematic and rule-governed**, not arbitrary — the same co-occurrence patterns reliably activate the same meaning domains across multiple glyphs.
2. **Creation context signal** (G610/G606 nearby) is the strongest — it overrides all other readings and activates cosmogonic meanings even for glyphs with stronger default non-cosmogonic readings.
3. **The voyage sequence** (G40→G8→G9) is the clearest three-glyph fixed phrase in the corpus — a fully readable navigation clause.
4. **Section start markers** (G32, G999) reset the reading context — the meaning of a glyph in section N may be completely different in section N+1.
5. The system is **learnable** by a trained reciter — the rules are consistent enough that someone who knew the oral traditions could read the tablets reliably without a phonetic key.
